# Poppins Font Files

This folder should contain the following Poppins font files for PDF generation:

## Required Files:
1. **Poppins-Regular.ttf**
2. **Poppins-Bold.ttf**

## How to Download:

### Option 1: Google Fonts (Recommended)
1. Go to: https://fonts.google.com/specimen/Poppins
2. Click "Download family"
3. Extract the ZIP file
4. Copy `Poppins-Regular.ttf` and `Poppins-Bold.ttf` to this folder

### Option 2: Direct Download
Download from: https://github.com/google/fonts/tree/main/ofl/poppins

## File Structure:
```
/public/fonts/
  ├── Poppins-Regular.ttf
  ├── Poppins-Bold.ttf
  └── README.md (this file)
```

## Note:
Once you add these files, the PDF receipts will automatically use Poppins font.
If the fonts are not found, the system will fallback to Helvetica.
